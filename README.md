api for portfolio <br/>

## installation:

```
git clone https://github.com/LonelySpaceDog/small-api/
cd small-api
npm install
```

## to run server:

```
npm start
```

Address to send requests to api:
[http://localhost:3000](http://localhost:3000)

## Additional scripts

`npm run debug`
To debug with ndb
