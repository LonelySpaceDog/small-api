api for portfolio <br/>

## Installation:

```
git clone https://github.com/LonelySpaceDog/small-api/
cd small-api
npm install
```
Also you may want to check config.env
## To run server:

```
npm start
```

Address to send requests to api:
[http://localhost:3000](http://localhost:3000)

## Additional scripts

`npm run debug`
To debug with ndb
